# Hibernate Product CRUD

## Description
This project demonstrates Hibernate CRUD operations and advanced HQL queries such as:
- Sorting
- Pagination
- Aggregate functions
- Filtering using WHERE and LIKE
- GROUP BY queries

## Technologies Used
- Java
- Hibernate
- MySQL
- Maven
- Eclipse / Spring Tool Suite

## Features Implemented
- Insert multiple Product records
- Sort products by price and quantity
- Pagination using HQL
- Aggregate queries (COUNT, MIN, MAX)
- Filter products using WHERE and LIKE
- GROUP BY description

## How to Run
1. Configure MySQL database
2. Update `hibernate.cfg.xml`
3. Run `MainApp.java`
